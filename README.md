# CIT-DEPT-DASHBOARD (ECE)
A professional Flutter application designed for the ECE Department at Coimbatore Institute of Technology to manage student academic points.

## 🚀 Features
* **Student View**: Real-time display of accumulated academic points.
* **Staff Dashboard**: Secure interface for staff to input and update points.
* **Firebase Backend**: Real-time data synchronization using Firestore.
* **Leadership Eligibility**: Automatic calculation of eligibility for Semester VI roles.

## 🛠️ Technical Stack
* **Frontend**: Flutter (Dart)
* **Backend**: Firebase Firestore & Auth
* **State Management**: Provider / setState
* **Storage**: GitHub (E: Drive Workspace)

## ⚙️ Installation
1. Clone the repository to your local machine.
2. Ensure Flutter is installed and `flutter doctor` is green.
3. Add your `google-services.json` to the `android/app/` folder.
4. Run `flutter pub get` and `flutter run`.# CIT-ECE-DASHBOARD
